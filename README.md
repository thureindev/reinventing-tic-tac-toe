## Making games by reinventing classic tic-tac-toe rules.

# How to use
1. Navigate to the twist-tac-toe folder.
   ```bash
   cd twist-tac-toe
   ```
2. Install vite.
   ```bash
   npm i vite
   ```
3. Run vite.
   ```bash
   npm run dev
   ```
4. Enjoy customizing your tic-tac-toe board. Have fun!
